# capitalize-word [![Build Status][travis-image]][travis-url]

> Capitalize each word

## Install

```sh
npm install --save capitalize-word
```

## Usage

```js
import capitalize from 'capitalize-word';

capitalize('hello world'); // Hello World
```

## API

### capitalize(input)

#### input

Type: `string`

Text for transform.
